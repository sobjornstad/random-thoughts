This is a publish-only repository for the public portions of Random Thoughts,
    Soren Bjornstad’s commonplace book.
See <https://randomthoughts.sorenbjornstad.com>.


## AI use

I have never used generative AI to write any part of the contents of Random Thoughts
and have no plans to.
I may use coding agents occasionally to maintain the related software tooling,
but that tooling is not in this repository.
